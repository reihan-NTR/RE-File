# FileShare - Platform Berbagi File

![FileShare Banner](https://via.placeholder.com/800x200?text=FileShare)

Platform berbagi file gratis dengan sistem verifikasi admin, real-time updates, dan database JSON.

## ✨ Fitur

- ✅ Upload file tanpa batasan ukuran
- ✅ Sistem verifikasi oleh admin
- ✅ Real-time notification dengan Socket.IO
- ✅ Edit dan hapus file oleh pemilik
- ✅ Sistem report file oleh user
- ✅ Pencarian dan filter file
- ✅ Database JSON (tanpa perlu MySQL)
- ✅ Responsive design

## 🛠️ Teknologi

- **Backend:** Node.js + Express
- **Database:** JSON File (self-contained)
- **Real-time:** Socket.IO
- **Frontend:** HTML5, CSS3, JavaScript
- **File Upload:** Multer

## 📦 Instalasi

```bash
# Clone repository
git clone https://github.com/username/file-sharing-app.git
cd file-sharing-app

# Install dependencies
npm install

# Jalankan server
npm run dev
